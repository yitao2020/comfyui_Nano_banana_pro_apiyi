# 多图像输入权重说明

## 功能说明

NanoBanana Pro 节点现在支持多张图像输入，并自动为每张图像设置不同的权重优先级。

## 权重优先级

当你使用批次组合节点（Batch Images）输入多张图像时：

### 图像 1（第一张）- 基础图 🎯
- **角色**：BASE IMAGE（要修改的目标图）
- **作用**：保持其主体和结构，但应用参考图的风格
- **建议用途**：放置你想要修改/风格化的原始图像

### 图像 2（第二张）- 最高权重 ⭐⭐⭐
- **角色**：PRIMARY STYLE REFERENCE（主要风格参考）
- **权重**：最高
- **作用**：AI 会强烈应用这张图的艺术风格、色调和视觉特征
- **建议用途**：放置你最想要的风格图像

### 图像 3（第三张）- 中等权重 ⭐⭐
- **角色**：SECONDARY REFERENCE（次要参考）
- **权重**：中等
- **作用**：作为额外的灵感和微妙影响
- **建议用途**：放置辅助风格或补充元素

### 图像 4+（第四张及以后）- 最低权重 ⭐
- **角色**：AUXILIARY（辅助参考）
- **权重**：最低
- **作用**：提供微妙的灵感
- **建议用途**：放置细节参考或氛围图

## 使用示例

### 示例 1：照片风格化

```
工作流：
LoadImage (原始照片) ──┐
LoadImage (油画风格) ──┼─→ Batch Images → NanoBanana Pro → SaveImage
LoadImage (色调参考) ──┘

说明：
- 图像 1：你的原始照片（保持主体）
- 图像 2：油画风格参考（最强影响）
- 图像 3：色调氛围参考（辅助影响）

提示词：将第一张照片转换为油画风格
```

### 示例 2：人物风格迁移

```
工作流：
LoadImage (人物照片) ──┐
                      ├─→ Batch Images → NanoBanana Pro → SaveImage
LoadImage (动漫风格) ──┘

说明：
- 图像 1：原始人物照片
- 图像 2：目标动漫风格（强烈应用）

提示词：保持人物特征，转换为动漫风格
```

### 示例 3：场景重绘

```
工作流：
LoadImage (原始场景) ──┐
LoadImage (主风格图) ──┼─→ Batch Images → NanoBanana Pro → SaveImage
LoadImage (次风格图) ──┘

说明：
- 图像 1：要修改的场景
- 图像 2：主要艺术风格（如赛博朋克）
- 图像 3：次要风格元素（如光影效果）

提示词：重新绘制场景，应用赛博朋克风格
```

## 技术实现

节点会自动：
1. 检测批次中的图像数量
2. 为每张图像添加权重说明文本
3. 按顺序发送给 API，第一张图像的权重描述最强

## 提示词建议

配合多图像输入时，提示词可以这样写：

**风格迁移**：
```
将第一张图片转换为第二张图片的艺术风格，保持原始构图和主体
```

**保持主体，改变风格**：
```
保持基础图的人物特征，应用参考图的绘画风格和色调
```

**场景重绘**：
```
重新绘制第一张图的场景，使用第二张图的风格和氛围
```

**细节控制**：
```
修改第一张图，强烈应用第二张图的风格，并融入第三张图的色彩元素
```

## 注意事项

1. **图像顺序很重要**：
   - 第一张：要修改的基础图
   - 第二张：主要风格参考（影响力最大）
   - 第三张：次要参考
   
2. **建议 2-3 张图像**：太多图像可能导致效果混乱

3. **图像会自动压缩**：每张图像都会压缩到 1024px 以避免请求过大

4. **Auto 模式**：选择 auto 宽高比时，会根据第一张图像（基础图）的比例自动选择

5. **提示词配合**：明确说明你想保持什么、改变什么，效果会更好

## 常见问题

**Q: 最多可以输入几张图像？**
A: 理论上没有限制，但建议 2-3 张。第一张是基础图，第二张是主要风格参考。

**Q: 如何调整权重？**
A: 权重是固定的：第一张=基础图，第二张=最高权重，第三张=中等权重。如果想改变，调整图像顺序。

**Q: 可以只用一张图像吗？**
A: 可以，直接连接单张图像即可，相当于图生图模式。

**Q: 第一张图会被完全改变吗？**
A: 不会，AI 会保持第一张图的主体和结构，但应用第二张图的风格。

**Q: 如果只想要风格迁移怎么办？**
A: 第一张放原图，第二张放目标风格图，提示词写"将第一张图转换为第二张图的风格"。
